## Edit Content

  - `/` → `app/page.tsx`
  - `/contact` → `app/contact/page.tsx`
  - `/impressum` → `app/impressum/page.tsx`
  - `/privacy` → `app/privacy/page.tsx`

- Global styles in `app/globals.css`.
- Tailwind config in `tailwind.config.ts`.
